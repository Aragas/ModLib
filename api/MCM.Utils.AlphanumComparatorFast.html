    Class AlphanumComparatorFast

    Alphanumeric sort. This sorting algorithm logically handles numbers in strings. It makes sense to humans. Highway names like 50F and 100F will be sorted wrongly with ASCII sort. It is different from alphabetic, ASCII or numeric sorting. This algorithmic approach is used in file managers.
